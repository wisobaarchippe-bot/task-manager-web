# Task Manager Web

## Description
Application web de gestion de tâches collaborative.  
Technologies : React (frontend), Node.js/Express (backend), MongoDB.

## Fonctionnalités
- Inscription / Connexion avec JWT
- Création et gestion des tâches
- Assignation des tâches à d'autres utilisateurs
- Statut des tâches (En cours, Terminé, Bloqué)
- Interface simple et réactive

## Installation

### Backend
```bash
cd backend
npm install
node server.js
